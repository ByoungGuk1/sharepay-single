# SharePay 정적 프로토타입

## 실행
- 이 폴더를 그대로 열고 `index.html`을 브라우저로 열면 됩니다.
- 로컬에서 링크가 `/...` 절대경로를 쓰기 때문에, 가장 편한 방법은 간단한 로컬 서버로 띄우는 것입니다.

### (선택) 로컬 서버
- Python:
  - `python -m http.server 5500`
  - 브라우저에서 `http://localhost:5500/` 접속

## 주요 페이지
- /index.html : 메인
- /notifications.html : 알림센터
- /wallet/personal/dashboard.html : 개인지갑
- /wallet/group/dashboard.html : 그룹지갑
- /community/index.html : 커뮤니티
- /my/profile.html : 마이페이지
- /styleguide.html : 디자인 시스템 체크
